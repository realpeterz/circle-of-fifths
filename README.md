# Circle of Fifths Interactive Keyboard

An elegant, interactive web application that visualizes the Circle of Fifths - a fundamental music theory tool showing the relationships between the 12 tones of the chromatic scale, their corresponding key signatures, and the associated major and minor keys.

## Features

- **Interactive Circle**: Click on any segment to explore different keys in the circle
- **Key Relationships**: See the selected key and its relative minor side by side
- **Chord Tones**: View the notes that make up the tonic chord
- **Intervals**: Identify the root, 3rd, and 5th for each key
- **Keyboard Support**: Use your computer keyboard to audition notes

## Musical Information Displayed

For each key, the application shows:
- **Key Name**: Major key (e.g., C Major)
- **Relative Minor**: The corresponding minor key (e.g., A Minor)
- **Chord Notes**: The notes that make up the tonic chord
- **Intervals**: The chord intervals (Root, 3rd, 5th)

## How to Use

1. **Click on any segment** in the circle to select a key
2. **Hover over segments** to preview them with visual feedback
3. **Use keyboard controls** (if available) to play notes
4. **Explore relationships** between different keys and their relative minors

## Educational Value

This tool is perfect for:
- Music students learning key signatures
- Composers exploring key relationships
- Musicians understanding chord progressions
- Anyone interested in music theory visualization
